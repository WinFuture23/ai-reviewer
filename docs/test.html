2. Text in das Feld unten eingeben (oder den Beispieltext lassen)
3. Unten rechts auf den blauen Tab "KI-Korrektor" klicken
4. "Artikel ueberpruefen" klicken
5. Debug-Log beobachten (im Widget: "Debug kopieren")
